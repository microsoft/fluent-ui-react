A themable React component library.

Stardust UI provides extensible JavaScript solutions for component state, styling, and accessibility. These powerful features are exposed behind simple API's based on natural language.

Stardust UI React is being built as an exemplar of the [Stardust UI Specifications](https://github.com/stardust-ui/specifications), component specifications, and utilities.
